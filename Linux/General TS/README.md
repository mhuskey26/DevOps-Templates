# General Linux Troubleshooting Scripts

These files are intentionally written as simple command sets you can copy and paste into a Linux shell.

Focus areas:
- Basic network health checks
- DNS troubleshooting
- DHCP troubleshooting
- Port and service validation

Notes:
- Most commands are read-only.
- Commands that need elevated access use `sudo`.
- Some tools may not be installed on every distro. Common alternates are included where possible.
